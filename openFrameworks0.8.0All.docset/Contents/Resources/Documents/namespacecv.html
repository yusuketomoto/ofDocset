class  Retina
 a wrapper class which allows the use of the Gipsa/Listic Labs retina model a wrapper class which allows the Gipsa/Listic Labs model to be used. This retina model allows spatio-temporal image processing (applied on still images, video sequences). As a summary, these are the retina model properties: => It applies a spectral whithening (mid-frequency details enhancement) => high frequency spatio-temporal noise reduction => low frequency luminance to be reduced (luminance range compression) => local logarithmic luminance compression allows details to be enhanced in low light conditions More...
