281  // matrix - vector multiplication
282  // although opengl uses premultiplication
283  // because of the way matrices are used in opengl:
284  //
285  // ofMatrix4x4 openGL
286  // [0] [1] [2] [3] [0] [4] [8] [12]
287  // [4] [5] [6] [7] [1] [5] [9] [13]
288  // [8] [9] [10] [11] [2] [6] [10] [14]
289  // [12] [13] [14] [15] [3] [7] [11] [15]
290  //
291  // in memory though both are layed in the same way
292  // so when uploading a matrix it just works without
293  // needing to transpose
294  //
295  // so although opengl docs explain transformations
296  // like:
297  //
298  // ofVec3f c = rotateZ 30º ofVec3f a around ofVec3f b
299  //
300  // openGL docs says: c = T(b)*R(30)*a;
301  //
302  // with ofMatrix4x4:
303  // ofMatrix4x4 R = ofMatrix4x4::newRotationMatrix(30,0,0,1);
304  // ofMatrix4x4 T = ofMatrix4x4::newTranlationMatrix(b);
305  // ofVec3f c = a*R*T;
306  // where * is calling postMult
