unsigned int  new_size

Set the number of objects in a list. If the size is increased, the new objects are not initialized in any way.

The allocation size formula is taken from Python's list: http://svn.python.org/view/python/trunk/Objects/listobject.c?view=markup

Objects may have moved after this call. Use svgtiny_list_get() to get new pointers.
